Detailed Description

Pseudo-legal move generation (quiet moves and captures).

This file generates pseudo-legal moves for each piece type. "Pseudo-legal" means: the moves follow piece movement rules, but may still leave the moving side in check.

The legality filtering is performed in evalpos.cpp (legalMoves()), which applies a move and rejects it if the king is left in check.

Current limitations:

  • No castling rights / castling moves are generated.
  • No en-passant capture is generated.
  • Promotions are generated as a single default choice (typically queen) unless expanded elsewhere.
