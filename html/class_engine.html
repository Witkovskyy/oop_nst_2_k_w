◆ applyMove()

void Engine::applyMove ( Board & board,
const Move & move,
Undo & undo )

Applies a move on the board (make/undo search loop).

Performs a lightweight "make move" used by the engine:

  • Updates board.squares[] (from-square becomes empty; to-square becomes moved piece),
  • Stores enough information into undo to revert later (captured piece, moved piece, promotion),
  • Updates incremental Zobrist key:
    • XOR out moved piece from from-square,
    • XOR out captured piece from to-square (if any),
    • XOR in moved/promoted piece on to-square,
    • XOR side-to-move key.
  • Handles promotion by changing the moved piece symbol to move.promotion.
Parameters
boardBoard state to mutate.
moveMove to apply.
undoOutput snapshot used later by undoMove().
Note
This function assumes Piece pointers live outside and are not owned here.

Negamax search with alpha-beta pruning and transposition table.

Negamax search with alpha-beta pruning and transposition table (TT).

Parameters
boardBoard state (mutated via apply/undo internally)
depthdepth in plies
alphaalpha bound
betabeta bound
colorPerspective sign (+1 white, -1 black)
Returns
best score

High-level flow: 1) Optional repetition detection (to avoid loops). 2) Transposition Table probe to reuse cached scores/bounds. 3) Terminal / depth cutoff:

  • depth == 0 -> quiescence()
  • game over / no legal moves -> mate/stalemate scoring 4) Generate legal moves, order them, recurse with sign flip: score = -negamax(child, depth-1, -beta, -alpha, -color) 5) Update alpha and prune when alpha >= beta. 6) Store result as TT_EXACT / TT_ALPHA / TT_BETA along with best move.

Quiescence search (captures only).

Capture-only extension search to reduce the horizon effect.

Parameters
boardBoard state (mutated via apply/undo internally)
alphaalpha bound
betabeta bound
colorPerspective sign (+1 white, -1 black)
Returns
quiescence score

Quiescence search continues exploring forcing capture sequences at depth 0 instead of returning a static evaluation immediately.

Algorithm:

  • Compute a "stand pat" evaluation.
  • If stand pat >= beta: fail-high cutoff.
  • Else try all capture moves: score = -quiescence(child, -beta, -alpha, -color)
  • Return best score within [alpha, beta].
Parameters
boardBoard state (mutated via apply/undo).
alphaAlpha bound.
betaBeta bound.
colorPerspective sign (+1 = White perspective, -1 = Black perspective).
Returns
Best tactical score in the quiescence search window.

Reverts a previously applied move (make/undo search loop).

Restores:

  • board.squares[] pointers (moved piece back to from-square, captured piece back to to-square),
  • piece positions (Piece::setPosition),
  • promotion (restores pawn symbol if promotion was applied),
  • incremental Zobrist hash updates (reverses exactly the operations in applyMove()).
