\doxysubsection{Detailed Description}
Pseudo-\/legal move generation (quiet moves and captures). 

This file generates pseudo-\/legal moves for each piece type. "{}\+Pseudo-\/legal"{} means:\+ the moves follow piece movement rules, but may still leave the moving side in check.

The legality filtering is performed in evalpos.\+cpp (legal\+Moves()), which applies a move and rejects it if the king is left in check.

Current limitations:\+
\begin{DoxyItemize}
\item No castling rights /\+ castling moves are generated.
\item No en-\/passant capture is generated.
\item Promotions are generated as a single default choice (typically queen) unless expanded elsewhere. 
\end{DoxyItemize}
